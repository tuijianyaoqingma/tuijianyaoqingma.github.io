### 第三步：写一个调用Gemini 2.0 Flash的Demo

下面这个类就是完整的调用代码，我尽可能让它保持简单、直接。你只需要修改代码前两行的 `API_KEY` 和 `MODEL` 参数，就能改成其他模型：

java
import okhttp3.*;
import com.alibaba.fastjson2.JSON;
import com.alibaba.fastjson2.JSONArray;
import com.alibaba.fastjson2.JSONObject;

import java.io.IOException;
import java.util.concurrent.TimeUnit;

public class GeminiViaYunWuAI {

    // 你的云雾API Key
    private static final String API_KEY = "sk-your-yunwu-api-key-here";
    // 模型名称，支持Gemini 2.0 Flash、1.5 Pro等
    private static final String MODEL = "gemini-2.0-flash-exp";
    // 云雾的API地址——国内可直接访问
    private static final String BASE_URL = "https://www.yunwuai.cc/v1";

    public static void main(String[] args) {
        OkHttpClient client = new OkHttpClient.Builder()
                .connectTimeout(60, TimeUnit.SECONDS)
                .readTimeout(60, TimeUnit.SECONDS)
                .writeTimeout(60, TimeUnit.SECONDS)
                .build();

        // 构建请求体：符合OpenAI chat completion标准
        JSONObject message = new JSONObject();
        message.put("role", "user");
        message.put("content", "用简体中文介绍一下Gemini 2.0 Flash模型的主要特点。");

        JSONArray messages = new JSONArray();
        messages.add(message);

        JSONObject requestBody = new JSONObject();
        requestBody.put("model", MODEL);
        requestBody.put("messages", messages);
        // 可选参数：控制温度、最大输出长度等
        requestBody.put("temperature", 0.7);
        requestBody.put("max_tokens", 1024);

        Request request = new Request.Builder()
                .url(BASE_URL + "/chat/completions")
                .addHeader("Authorization", "Bearer " + API_KEY)
                .post(RequestBody.create(requestBody.toJSONString(), MediaType.parse("application/json")))
                .build();

        try (Response response = client.newCall(request).execute()) {
            if (!response.isSuccessful()) {
                System.out.println("请求失败，HTTP状态码: " + response.code());
                System.out.println("错误详情: " + response.body().string());
                return;
            }

            String responseBody = response.body().string();
            JSONObject result = JSON.parseObject(responseBody);

            // 从标准OpenAI格式的响应中提取文本
            String content = result.getJSONArray("choices")
                    .getJSONObject(0)
                    .getJSONObject("message")
                    .getString("content");

            System.out.println("===== Gemini 2.0 Flash 返回结果 =====");
            System.out.println(content);
            System.out.println("======================================");

            // 调试用：打印完整的响应结构（包含Token消耗情况）
            System.out.println("完整响应JSON（含Token用量）: ");
            System.out.println(result.toJSONString());

        } catch (IOException e) {
            System.err.println("调用期间发生网络异常: " + e.getMessage());
            e.printStackTrace();
        }
    }
}


运行这段代码，如果一切正常，控制台会直接打印出Gemini 2.0 Flash用简体中文回复的内容。整个过程没有任何代理配置，就是写代码、运行、看结果。
