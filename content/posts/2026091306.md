## 第二步：写核心代码（纯纯的 Node.js，不花哨）

创建一个 `qwen_vl_demo.mjs` 文件：

javascript
import OpenAI from "openai";

const client = new OpenAI({
  baseURL: "https://www.yunwuai.cc/v1",
  apiKey: "sk-你的云雾API中转站Key", // 替换成你的key
});

import fs from "fs";

async function main() {
  // 读取本地图片并转成 Base64
  const imageBuffer = fs.readFileSync("./test.jpg");
  const base64Image = imageBuffer.toString("base64");

  const response = await client.chat.completions.create({
    model: "qwen-vl-plus",  // 推荐使用 qwen-vl-plus，兼顾速度与效果
    messages: [
      {
        role: "user",
        content: [
          { type: "text", text: "请详细描述这张图片里包含了什么。" },
          {
            type: "image_url",
            image_url: {
              url: `data:image/jpeg;base64,${base64Image}`,
            },
          },
        ],
      },
    ],
    max_tokens: 1000,
  });

  console.log(response.choices[0].message.content);
}

main();


**避坑提示：**
- 图片转 Base64 时，`data:image/jpeg;base64,` 这个前缀不能丢，否则 Qwen-VL 无法识别。
- 如果你用的是 PNG 图片，前缀要改成 `data:image/png;base64,`。
- `model` 字段建议使用 `qwen-vl-plus`，它是最新且兼容性最好的视觉模型。不要用旧的 `qwen-vl`，可能会报错不支持某些调用格式。
