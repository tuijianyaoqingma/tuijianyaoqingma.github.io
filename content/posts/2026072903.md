## Java接入实战：改一行代码，跑通GPT-4

下面我会用最常用的Java HTTP客户端（OkHttp）和JSON处理库（Gson）来演示，步骤极其简单：

### 前提条件

### 核心代码：调用GPT-4o

java
import okhttp3.*;
import com.google.gson.Gson;
import com.google.gson.JsonObject;

import java.io.IOException;

public class YunwuOpenAIExample {

    private static final String API_KEY = "你的云雾API Key"; // 替换
    private static final String BASE_URL = "https://www.yunwuai.cc/v1";
    private static final MediaType JSON = MediaType.get("application/json");

    public static void main(String[] args) throws IOException {
        OkHttpClient client = new OkHttpClient();

        // 构造请求体
        JsonObject message = new JsonObject();
        message.addProperty("role", "user");
        message.addProperty("content", "用Java写一个简单的快速排序");

        JsonObject requestBody = new JsonObject();
        requestBody.addProperty("model", "gpt-4o"); // 也可以换成 gpt-4o-mini, claude-3-5-sonnet 等
        requestBody.add("messages", new com.google.gson.JsonArray());
        requestBody.getAsJsonArray("messages").add(message);

        // 构建HTTP请求
        Request request = new Request.Builder()
                .url(BASE_URL + "/chat/completions")
                .addHeader("Authorization", "Bearer " + API_KEY)
                .post(RequestBody.create(requestBody.toString(), JSON))
                .build();

        // 发送请求并获取响应
        try (Response response = client.newCall(request).execute()) {
            if (!response.isSuccessful()) {
                System.out.println("请求失败: " + response.code() + " " + response.message());
                return;
            }
            String responseBody = response.body().string();
            // 解析JSON
            JsonObject jsonResponse = new Gson().fromJson(responseBody, JsonObject.class);
            String content = jsonResponse.getAsJsonArray("choices")
                    .get(0).getAsJsonObject()
                    .get("message").getAsJsonObject()
                    .get("content").getAsString();
            System.out.println("AI回复:\n" + content);
        }
    }
}


**看到没有？** 唯一的改动就是把`base_url`从`https://api.openai.com/v1`换成`https://www.yunwuai.cc/v1`，API key换成云雾的。模型名称用OpenAI官方命名完全一致，代码里甚至可以用`gpt-4o`、`claude-3-5-sonnet`等非OpenAI模型——云雾会自动路由到对应模型。

## 进阶用法：流式响应（Stream）

很多场景（如聊天机器人）需要实时显示AI回复，Java配合OkHttp流式读取非常顺畅：

java
// 构建相同请求，但增加stream: true 参数
requestBody.addProperty("stream", true);

Request request = new Request.Builder()
        .url(BASE_URL + "/chat/completions")
        .header("Authorization", "Bearer " + API_KEY)
        .post(RequestBody.create(requestBody.toString(), JSON))
        .build();

OkHttpClient client = new OkHttpClient.Builder()
        .readTimeout(60, java.util.concurrent.TimeUnit.SECONDS)
        .build();

try (Response response = client.newCall(request).execute()) {
    if (!response.isSuccessful()) {
        System.out.println("请求失败: " + response);
        return;
    }
    // 读取流式数据
    ResponseBody body = response.body();
    if (body != null) {
        java.io.InputStream in = body.byteStream();
        java.util.Scanner scanner = new java.util.Scanner(in);
        while (scanner.hasNextLine()) {
            String line = scanner.nextLine();
            if (line.startsWith("data: ")) {
                String data = line.substring(6);
                if ("[DONE]".equals(data)) {
                    break;
                }
                // 解析JSON，提取content增量
                JsonObject delta = new Gson().fromJson(data, JsonObject.class);
                String contentDelta = delta.getAsJsonArray("choices")
                        .get(0).getAsJsonObject()
                        .get("delta").getAsJsonObject()
                        .get("content").getAsString();
                System.out.print(contentDelta); // 实时打印
            }
        }
        scanner.close();
    }
}


流式接入后，你的Java应用就可以像真正的AI聊天一样逐字显示回复。
