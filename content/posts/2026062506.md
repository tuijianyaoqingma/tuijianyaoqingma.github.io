## Llama 4 Java SDK 集成方案

下面直接上干货。分三部分：环境准备、核心代码、完整避坑清单。

### 1. 环境准备

在 Java 项目中引入 OpenAPI 官方 Java SDK 的 Maven 依赖（Llama 4 接口兼容 OpenAI 格式，所以直接复用此 SDK）：

xml
<dependency>
    <groupId>com.theokanning.openai-gpt3-java</groupId>
    <artifactId>service</artifactId>
    <version>0.18.2</version>
</dependency>


如果你使用的是 Gradle，在 `build.gradle` 中添加：

gradle
implementation 'com.theokanning.openai-gpt3-java:service:0.18.2'


### 2. 核心代码示例

以下是一个可直接运行的 Llama 4 对话请求示例，代码注释清晰，拷贝即用：

java
import com.theokanning.openai.OpenAiService;
import com.theokanning.openai.completion.chat.ChatCompletionRequest;
import com.theokanning.openai.completion.chat.ChatMessage;
import com.theokanning.openai.completion.chat.ChatMessageRole;

import java.time.Duration;
import java.util.ArrayList;
import java.util.List;

public class Llama4ChatExample {

    public static void main(String[] args) {
        // 1. 设置 API Key（从[[云雾ai官网](https://www.yunwuai.cc/)](https://www.yunwuai.cc/)后台获取）
        String apiKey = "你的云务apiKey";

        // 2. 修改 baseUrl 为[[云雾ai官网](https://www.yunwuai.cc/)](https://www.yunwuai.cc/)的国内直连地址
        String baseUrl = "https://www.yunwuai.cc/v1";

        // 3. 创建 OpenAiService 实例，设置超时时间（建议至少 30 秒）
        OpenAiService service = new OpenAiService(apiKey, Duration.ofSeconds(60));

        // 4. 构建消息列表
        List<ChatMessage> messages = new ArrayList<>();
        messages.add(new ChatMessage(ChatMessageRole.SYSTEM.value(), "你是一个有用的AI助手。"));
        messages.add(new ChatMessage(ChatMessageRole.USER.value(), "请用中文介绍一下Llama 4模型的特色。"));

        // 5. 设置请求参数
        ChatCompletionRequest request = ChatCompletionRequest.builder()
                .model("llama4")                 // 指定模型为 Llama 4
                .messages(messages)
                .maxTokens(1024)                 // 最大输出长度
                .temperature(0.7)                 // 创意度控制
                .stream(false)                    // 是否流式输出（这里使用非流式）
                .build();

        try {
            // 6. 发送请求并获取结果
            String response = service.createChatCompletion(request)
                    .getChoices().get(0).getMessage().getContent();
            System.out.println("Llama 4 回复：");
            System.out.println(response);
        } catch (Exception e) {
            System.err.println("请求失败，请检查网络或API Key：" + e.getMessage());
        } finally {
            // 7. 关闭服务连接
            service.shutdownExecutor();
        }
    }
}


### 3. 流式输出示例（适合实时聊天场景）

java
import com.theokanning.openai.completion.chat.ChatCompletionChunk;
import io.reactivex.Flowable;

// ...（重复上述步骤1-5，但将 stream 设置为 true）

ChatCompletionRequest request = ChatCompletionRequest.builder()
        .model("llama4")
        .messages(messages)
        .maxTokens(1024)
        .temperature(0.7)
        .stream(true)
        .build();

Flowable<ChatCompletionChunk> flowable = service.streamChatCompletion(request);

System.out.println("流式回复：");
flowable
    .doOnError(Throwable::printStackTrace)
    .forEach(chunk -> {
        chunk.getChoices().stream()
                .filter(choice -> choice.getMessage() != null)
                .map(choice -> choice.getMessage().getContent())
                .filter(content -> content != null)
                .forEach(System.out::print);
    });

## 完整避坑清单

以下是真实踩坑经验，帮你绕过那些可能浪费半天时间的隐形问题：

| 问题 | 原因 | 解决方式 |
|------|------|----------|
| **Connection Timeout**（连接超时） | 网络代理没关干净，或 `baseUrl` 地址错误 | 确保 `baseUrl` 设置为 `https://www.yunwuai.cc/v1`，并关闭系统代理 |
| **401 Unauthorized** | API Key 无效或格式错误 | 重新从[[云雾ai官网](https://www.yunwuai.cc/)](https://www.yunwuai.cc/)后台拷贝 Key，确认没多空格或换行 |
| **Model Not Found** | 模型名称写错了 | 在云雾模型列表确认 Llama 4 的准确模型 ID（例如 `llama4`） |
| **Request Timeout** | 请求超时时间设置过短 | 将 `Duration.ofSeconds(60)` 改为 `Duration.ofSeconds(120)` |
| **Token 余额不足** | 免费额度用完了没充值 | 先确认账号余额，最少充 1 元就能继续用 |
| **SSLHandshakeException** | JDK 版本过旧，TLS 协议不兼容 | 升级 JDK 到 11+，或配置 TLSv1.2 |
| **流式输出乱码** | 编码未设置 | JVM 启动参数加上 `-Dfile.encoding=UTF-8` |
