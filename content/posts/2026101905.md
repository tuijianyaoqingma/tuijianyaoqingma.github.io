## 错误排查：遇到报错怎么办

**1. 返回 401 Unauthorized**

检查 API Key 是否正确。去[云雾API中转站后台](www.yunwuai.cc)重新复制一次，确认没有多余空格。

**2. 返回 400 Bad Request - model not found**

确认模型名写的是 `claude-3-opus-20240229`。不同分组的模型名可能略有差异，以平台文档为准。如果你使用的是限时特价分组，模型名会有所不同。

账户余额不足或并发超限。云雾API中转站的并发没有硬性限制，但如果没有余额会触发限流。去后台充值再试，最低 1 元就能充。

确认你用的是 `https://www.yunwuai.cc/v1` 而不是其他地址。云雾API中转站有多个全球节点，国内直连应该没有问题。如果仍然超时，检查是否有防火墙拦截，或者尝试重启路由器。

## 高级用法：流式输出 + 长上下文

Claude Opus 的真正优势在于**长上下文处理**。雾API中转站支持 Opus 的 200K 上下文窗口，这对做文档分析、代码库解读非常有用。

下面是带**流式输出**的代码示例，适合交互式应用场景：

javascript
// stream-demo.mjs
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: '你的云雾API中转站API Key',
  baseURL: 'https://www.yunwuai.cc/v1',
});

async function main() {
  const stream = await client.chat.completions.create({
    model: 'claude-3-opus-20240229',
    messages: [
      { role: 'user', content: '解释一下Node.js的事件循环机制，要求详细但不啰嗦，控制在1000字以内' }
    ],
    max_tokens: 2048,
    stream: true, // 开启流式输出
  });

  for await (const chunk of stream) {
    const content = chunk.choices[0]?.delta?.content || '';
    process.stdout.write(content);
  }
}

main();


运行这个代码，你会看到 Claude Opus 的内容逐步打印出来，而不是一次性返回。这种方式适合用在实时聊天、代码提示、长文档生成等场景。
