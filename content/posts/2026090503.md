## Java接入o3-mini：最直白的示例代码

说实话，接入过程真的只是改几行代码的事。无论你用的是Spring Boot、Apache HttpClient还是Plain Java，核心逻辑都一样。

下面是一个最标准的Java示例，直接调用o3-mini接口：

java
import okhttp3.*;
import org.json.JSONObject;
import java.io.IOException;

public class O3MiniExample {
    // 核心：替换base_url和api_key
    private static final String API_URL = "https://www.yunwuai.cc/v1/chat/completions";
    private static final String API_KEY = "你的云雾API密钥"; // 去www.yunwuai.cc注册获取

    public static void main(String[] args) throws IOException {
        OkHttpClient client = new OkHttpClient();

        JSONObject json = new JSONObject();
        json.put("model", "o3-mini"); // 直接指定模型
        json.put("messages", new JSONObject[]{
            new JSONObject().put("role", "user").put("content", "用Java写一个快速排序算法。")
        });
        json.put("max_completion_tokens", 2000); // o3-mini支持

        RequestBody body = RequestBody.create(
            json.toString(), MediaType.get("application/json; charset=utf-8"));

        Request request = new Request.Builder()
            .url(API_URL)
            .addHeader("Authorization", "Bearer " + API_KEY)
            .post(body)
            .build();

        Response response = client.newCall(request).execute();
        System.out.println(response.body().string());
    }
}


**和OpenAI官方代码的差异：**
1.  将 `base_url` 从 `https://api.openai.com/v1` 改成 `https://www.yunwuai.cc/v1`。
2.  把OpenAI的API Key换成在[云雾api聚合站](https://www.yunwuai.cc/)申请的Key。
3.  模型指定为 `o3-mini`。

就这么简单。你的LangChain、LlamaIndex、或者其他Java框架，基本只需要修改配置文件里的API基础地址。
