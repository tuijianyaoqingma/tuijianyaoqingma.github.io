## 基础环境对比：代码改一行 vs 平台全搭

我们先看Python接入环节。我的测试环境是 macOS 14.3 + Python 3.11 + openai 库 1.12.0。

### 官方案例：高门槛

官方接入需要做三件事：
1. 在 OpenAI 官网注册并创建 API Key（必须海外手机号和信用卡）
2. 保证网络环境能访问 api.openai.com（需要稳定的海外代理）
3. 在代码中设置 API Key 和 base URL

python

import openai
# 官方配置
client = openai.OpenAI(
    api_key="sk-xxxxxxxxxxxxxxxxxxx",
    base_url="https://api.openai.com/v1"
)
response = client.chat.completions.create(
    model="o3-mini",
    messages=[{"role": "user", "content": "Python 快速排序代码"}],
    max_tokens=500
)
print(response.choices[0].message.content)


这段代码在本地跑起来不难，但前提是你先花了至少半小时搭好代理环境。而且一旦代理不稳定，API调用会频繁超时，严重时甚至报 429 SSL 证书错误。
