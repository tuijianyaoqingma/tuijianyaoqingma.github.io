## 二、防封号技巧：为什么你老被封？

很多新手折腾 Doubao API 最大的痛点，就是封号。一旦被封，账号里充的钱、写的代码全都作废。我研究了平台策略，结合一些老手的经验，整理出下面“保命三招”。

### 防封招数2：控制调用频率，不要狂轰滥炸

很多人一拿到 Key，就想测试它的极限，用代码循环1000次发送请求。这种“短时间、高并发”的攻击行为，是封号的首要原因。豆包官方对并发量比较敏感。

如果你每次调用 API 的 IP 地址都不同（比如一会儿用公司的网，一会儿用手机热点，一会儿又用代理），这种“中国漂移”在网络请求里是很可疑的。
